1. Install AWS CLI (if not installed)

2. Configure AWS CLI Profile
aws configure --profile my-profile-name
aws configure --profile amer
aws configure --profile root



    AWS Access Key ID [None]: <your-access-key-id>
    AWS Secret Access Key [None]: <your-secret-access-key>
    Default region name [None]: us-east-1
    Default output format [None]: json

3. Check the AWS Configuration
The AWS CLI will save your credentials to two files on your system:

Configuration file (~/.aws/config for Linux/macOS 
                    or 
                C:\Users\<your-username>\.aws\config for Windows)

Credentials file (~/.aws/credentials for Linux/macOS 
                    or 
                C:\Users\<your-username>\.aws\credentials for Windows)
